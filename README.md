# 💼 Salary Prediction Pro

An end-to-end Machine Learning project to predict salaries based on experience and skill level, with an interactive web application using Streamlit.

---

## 📌 Overview

This project explores how different factors affect salaries and builds a predictive model using multiple machine learning algorithms.

---

## 🚀 Features

✅ Salary prediction based on user input  
✅ Model comparison (Linear Regression, Random Forest, Decision Tree)  
✅ Interactive dashboard with visualizations  
✅ Feature importance analysis  
✅ Upload your own dataset and get predictions  

---

## 🤖 Models Used

- Linear Regression
- Random Forest
- Decision Tree

---

## 📊 How It Works

1. Load dataset  
2. Train multiple models  
3. Compare performance (R², MAE)  
4. Select best model  
5. Predict salary based on input
